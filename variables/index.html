Before ES6, we only had one way to delcare a variable - VAR
Now we can use CONST and LET.
Var is function scope but CONST and LET are block scope


VAR

We can define, update and redefine this method of defining a variable .

LET

Again, we can define and update with the LET method, however we cannot redefine like we can with VAR.

CONST

As the term suggests, we use CONST for data than will never change. For example, boiling point of water, PI, freezing point of water...

However, if it is a reference type (see example), then we can change the data.
